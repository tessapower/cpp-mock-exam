## Exam Modules

1. **[Sequence Containers and Container Adapters](./module-1-sequence-containers.md)**
   - `std::vector`, `std::deque`, `std::list`, `std::forward_list`, `std::array`
   - Container adapters: `std::stack`, `std::queue`, `std::priority_queue`

2. **[Associative Containers](./module-2-associative-containers.md)**
   - `std::set`, `std::multiset`, `std::map`, `std::multimap`
   - `std::unordered_set`, `std::unordered_map`

3. **[Non-Modifying Sequence Operations](./module-3-non-modifying-operations.md)**
   - `std::find`, `std::count`, `std::for_each`, `std::search`, `std::equal`

4. **[Modifying Sequence Operations](./module-4-modifying-operations.md)**
   - `std::copy`, `std::transform`, `std::remove`, `std::replace`, `std::reverse`, `std::rotate`

5. **[Sorting and Binary Search](./module-5-sorting-algorithms.md)**
   - `std::sort`, `std::stable_sort`, `std::binary_search`, `std::lower_bound`, `std::upper_bound`

6. **[Merge, Heap, Min, Max Algorithms](./module-6-merge-heap-algorithms.md)**
   - `std::merge`, `std::inplace_merge`, set operations, `std::min_element`, `std::max_element`

7. **[STL Functional Objects and Utilities](./module-7-functional-utilities.md)**
   - Function objects, `std::bind`, `std::function`, `std::mem_fn`

8. **[Advanced I/O](./module-8-advanced-io.md)**
   - Stream manipulators, file I/O, string streams, formatting

9. **[Templates](./module-9-templates.md)**
   - Function templates, class templates, specialization, variadic templates
